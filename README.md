# 🌦 Weather App

A modern **Weather Application** built using **React + Tailwind CSS** that provides real-time weather data, dynamic UI, and location-based forecasts.

---

## 🚀 Features

- 🌍 Search weather by city
- 📍 Get weather using current location
- 🌙 Dark / Light mode toggle
- 🌈 Dynamic background based on weather
- 🌤 Weather icons (Clear, Clouds, Rain, Snow)
- ⏳ Loading indicator
- 📊 Hourly forecast preview
- 💅 Modern glassmorphism UI

---

## 🛠 Tech Stack

- ⚛️ React (Vite)
- 🎨 Tailwind CSS
- 🌐 OpenWeather API
- 🚀 Vercel (Deployment)
